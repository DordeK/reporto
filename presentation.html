Security at Every Layer

🔒 ALLOWED_FIELDS Allowlist

30+ whitelisted columns. LLM hallucinations are blocked before SQL is compiled.

🛡️ Parameterised SQL Only

SQLAlchemy text() with named parameters — zero string interpolation.

✅ EXPLAIN Before Execute

PostgreSQL validates every query plan before it touches real data.

📊 Reconciliation + Quality Score

Report totals cross-checked against DB sums. Anomaly density → 0–100 score per report.

📉 Works Without OpenAI

4 keyword templates cover core reports — zero hard dependency on AI.

+ Drill-Down Mode

Click any report result row to drill into the underlying invoices — filters and grouping applied automatically.
